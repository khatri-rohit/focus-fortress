## Extension for productivity

### Block Websites and Apps while working

- This extension will block websites and apps while I coding or working on my computer.
- It's my own extension and I'm not affiliated with any of the websites or apps.

### Features

- Block websites and apps while I coding or working on my computer.

### How to use

## Prerequisites

- Vs Code
- Install the Wakatime plugin for VS Code.

## Installation and Usage

- Install the extension from the [Chrome Web Store](https://chrome.google.com/webstore/detail/productivity-ext/nitin-rohit-productivity-e/publish-accepted?authuser=0&hl=en-US&gl=US).
- Click on the extension icon to open the options page.
- It shows the Current Opened Tab URL and the Blocked Websites and Apps on click on block button it will block the website when ur coding.
